Gestión de proyectos y etiquetas

Aprende a organizar tus productos y gestionar el ciclo de revisión de etiquetas paso a paso.

¿Qué es un proyecto?

Un proyecto es el contenedor principal que agrupa uno o varios productos. Puedes pensar en él como una carpeta que representa un objetivo de negocio: un lanzamiento, una línea de productos o un mercado destino.

Cada proyecto puede estar en uno de estos estados:

  • Activo — Estás trabajando activamente en él.
  • Pausado — Detenido temporalmente, pero conserva toda su información.
  • Cerrado — El proyecto fue concluido.
  • Archivado — Se mueve fuera de la vista principal sin eliminar nada.

Cómo crear un proyecto

  1. Ve a Proyectos en el menú lateral.
  2. Haz clic en "Nuevo proyecto".
  3. Ingresa el nombre y una descripción opcional.
  4. Confirma para crearlo. Serás redirigido al detalle del proyecto.

Tu plan determina cuántos proyectos activos puedes tener simultáneamente. Puedes ver el límite en el Dashboard o en Suscripción.

Cómo agregar productos

Dentro de un proyecto, puedes registrar cada producto que requiere revisión de etiqueta:

  1. Abre el proyecto al que quieres agregar el producto.
  2. Haz clic en "Agregar producto".
  3. Completa el nombre, categoría, país de destino y descripción del producto.
  4. Guarda el producto para acceder a su sección de etiquetas.

Cómo subir una etiqueta para revisión

  1. Abre el producto al que pertenece la etiqueta.
  2. Haz clic en "Subir etiqueta" o "Nueva versión".
  3. Arrastra tu archivo o selecciónalo desde tu equipo (PDF, PNG, JPG, DOCX — máximo 20 MB).
  4. Agrega notas opcionales para el revisor (p. ej. puntos específicos a verificar).
  5. Haz clic en "Enviar para revisión". La etiqueta quedará en estado Enviada.
Versiones

Cada vez que corriges y subes el archivo, se crea una nueva versión. El historial completo de versiones queda disponible en el detalle de la etiqueta para que tanto tú como el revisor puedan comparar cambios.

Entendiendo la respuesta del revisor

Cuando el revisor termina el análisis, recibirás una notificación y la etiqueta pasará a uno de estos estados:

✓
Aprobada

Tu etiqueta cumple con la normativa. Puedes avanzar a los trámites oficiales.

!
Cambios requeridos

El revisor encontró observaciones. Revisa las notas y documentos adjuntos, corrige tu etiqueta y sube una nueva versión.

Consejos para acelerar la aprobación

  • Asegúrate de que el archivo sea legible y esté a buena resolución (mín. 300 dpi para imágenes).
  • Usa el campo de notas para indicar si hay elementos específicos que el revisor deba verificar.
  • Revisa las observaciones anteriores antes de subir una nueva versión para no repetir el mismo error.
  • Consulta la normativa del país de destino antes de enviar (puedes abrir una consultoría desde el menú).
